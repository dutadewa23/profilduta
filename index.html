Deskripsi Project

Project yang dibuat untuk kebutuan pertanian ditengah kota yang minim ruang terbuka. Project ini dibuat menggunakan arduino uno sebagai 'otak' alat tersebut. Input daya yang digunakan adalah 220V AC 50Hz dan outputnya adalah DC 12V yang akan menyalakan komponen seperti pompa dan lampu UV. Alat ini juga menggunakan timer untuk mengatur ritme penyiraman dan pencahayaan

Fitur-fitur utama alat ini meliputi, Sensor kelembaban untuk mendeteksi kelembaban tanah, Sensor Cahaya untuk mendeteksi pencahayaan dalam ruangan dan Sensor Suhu untuk mendeteksi suhu tanah. ALat ini menggabungkan banyak data dari sensor tersebut dan membuat keputusan dari algoritma yang telah dibuat.

Project ini merupakan pembelajaran mandiri yang diperuntukan untuk mengetahui bagaimana sebuah siklus berulang dapat disematkan teknologi otomasi yang dapat meningkatkan efisiensi pekerjaan.
